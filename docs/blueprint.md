# **App Name**: TiempoDiario

## Core Features:

- Captura Rápida de Tareas: Una interfaz de usuario clara que pregunta '¿Qué quieres hacer hoy?' y permite la creación rápida de tareas con nombre y duración estimada.
- Gestión de Tareas: Permite a los usuarios crear, leer, actualizar y eliminar tareas, incluyendo la asignación de una duración y la vinculación a una tarea 'Padre'. Los datos de las tareas se persisten en una base de datos.
- Programación Visual de Bloques: Una interfaz intuitiva de arrastrar y soltar para organizar tareas como bloques de tiempo en una línea de tiempo diaria.
- Temporizador de Enfoque Visual: Un overlay de interfaz de usuario que visualiza el bloque de tiempo de la tarea actual con una flecha animada que actúa como un temporizador, mostrando el progreso de la tarea.
- Exportación de Plan de Trabajo: Permite al usuario exportar el horario de tareas planificado y los tiempos asignados a un archivo CSV para análisis externos.

## Style Guidelines:

- Esquema de color claro. El color primario es un cian profesional y calmado (#3495C0) para denotar concentración y eficiencia.
- El color de fondo es un azul-gris muy claro y desaturado (#EBF3F6), que complementa el primario sin distraer y es ideal para largos períodos de concentración.
- Un color de acento es un turquesa suave (#66CCBD) para resaltar elementos importantes, CTAs o el progreso en el temporizador visual.
- La tipografía recomendada para títulos y cuerpo es 'Inter' (sans-serif), elegida por su claridad, legibilidad neutra y modernidad, que se alinea con una aplicación de productividad enfocada.
- Utilizar iconos lineales y modernos para acciones como añadir, editar, arrastrar, iniciar temporizador y exportar, manteniendo la interfaz limpia y fácil de entender.
- Un diseño de cuadrícula o línea de tiempo limpio y estructurado para la vista diaria, permitiendo que los bloques de tareas sean claramente distinguibles y fáciles de manipular. El espacio en blanco es clave para evitar la sobrecarga visual.
- Animaciones sutiles y fluidas para acciones de arrastrar y soltar tareas, transiciones de estado de tareas (activo/completado) y actualizaciones del temporizador visual, mejorando la retroalimentación del usuario sin causar distracción.
